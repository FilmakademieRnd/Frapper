Shading demo Scene



Introduction

This demo scene demonstrates the shading and compositing capabilities of Frappper. You can view each node's result by activating its view flag. You can pan, rotate, zoom the view using ALT + MMB, ALT + RMB, ALT + LMB. Please feel encouraged to play with all parameters in the Parameter panel to see a change in behavior.



Structure

AnimatableMesh

AnimatableMesh nodes can load Ogre .mesh and .skeleton files. This scene contains two meshes: a low poly car mesh and a high poly dragon mesh.



Transform

Transform nodes allow to transform any input geometry. They can also be used to merge geometry.



Camera

Camera nodes are used to render certain geometry using the AdvancedRender node.



AdvancedRender

The AdvancedRender node allows to render to an off screen render target. It outputs a color target, normal target, depth target and view target, which can be incorporated to do deferred shading further down the pipeline. Geometry, cameras and lights can be connected as inputs.



SimpleBlend

SimpleBlend nodes can be used to blend two input targets. Available blend modes are: add, mul, and A over B. The three blend nodes right after the AdvancedRender node are used to display the normal, depth and view targets.



SurfaceCurvature and Lightwarping

These two nodes implement a lightwarping algorithm which has been presented at SIGGRAPH 2009. For further information please visit: Lightwarping



AmbientOcclusion and GeometryBlur

These two nodes implement a screen space ambient occlusion approach.
